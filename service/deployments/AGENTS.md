# deployments — Скрипты деплоя

## Назначение

Директория содержит скрипты сборки, установки и запуска приложений проекта, организованные по приложениям и платформам.

## Структура

```
deployments/
└── go-draft/        # Деплой приложения (см. go-draft/AGENTS.md)
    └── linux/       # Linux amd64: build, install (см. linux/AGENTS.md)
```

## Приложения

| Приложение | Платформы | Makefile-команды |
|------------|-----------|------------------|
| `go-draft` | Linux amd64 | `build-app`, `install-app`, `uninstall-app` |

## Правила

- Каждое приложение — отдельная поддиректория с именем приложения
- Внутри — поддиректории по платформам (`linux/`, `darwin/` и т.д.)
- Makefile используют `include` из `configs/_make_/` для переменных сборки
